# 2103.11528 - Baselines and Training Setup (ViHSD)

- Paper: **A Large-scale Dataset for Hate Speech Detection on Vietnamese Social Media Texts**
- arXiv: https://arxiv.org/abs/2103.11528
- DOI: 10.1007/978-3-030-79457-6_35
- PDF: `2103.11528v4.pdf`

## Experimental settings reported (Section 5.1)
- Preprocessing: word segmentation (pyvi), stopword removal, lowercasing (except cased models), remove hashtags/urls/mentions.
- Text-CNN: epochs=50, batch=256, seq_len=100, dropout=0.5, Conv2D filters=32 with kernel sizes 2/3/5.
- GRU: epochs=50, seq_len=100, dropout=0.5, bidirectional GRU.
- Optimizer for Text-CNN/GRU: Adam.
- Transformers (BERT/XLM-R/DistilBERT): batch=16 (train/eval), epochs=4, seq_len=100, seed=4.

## Baseline results on ViHSD test set (Table 4)
- Text-CNN (fastText): Accuracy 86.69, F1-macro 61.11
- GRU (fastText): Accuracy 85.41, F1-macro 60.47
- BERT (bert-base-multilingual-uncased): Accuracy 86.60, F1-macro 62.38
- BERT (bert-base-multilingual-cased): Accuracy 86.88, F1-macro **62.69** (best in this paper)
- XLM-R (xlm-roberta-base): Accuracy 86.12, F1-macro 61.28
- DistilBERT (distilbert-base-multilingual-cased): Accuracy 86.22, F1-macro 62.42
